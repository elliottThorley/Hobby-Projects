This is a WIP 2D Platformer

Currently has partial KBM support and Gamepad support

Features:
	-main menu
	-save game system
	-character controller
	-basic level
	-pause menu
	-gamepad support
	-kbm support for menu navigation, kbm support for character controller coming soon

Controls:
	KBM:
	-W and S for navigating menu, enter to select
	-ESC for pause

	Gamepad:
	-DPad for navigating menu, X to select
	-Left Joystick for character movement
	-Left Trigger for wall climb
	-Left Shoulder for jump/wall jump
	-Start for pause
